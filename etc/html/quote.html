blockqoute tag & cite tag

HTML <blockquote> 요소는 안쪽의 텍스트가 긴 인용문임을 나타냅니다.
주로 들여쓰기를 한 것으로 그려집니다. (외형을 바꾸는 법은 사용 일람을 참고하세요)
인용문의 출처 URL은 cite 특성으로, 출처 텍스트는 <cite> 요소로 제공할 수 있습니다.

p tag for short quote

q 태그에 대해 MDN 문서는 HTML <q>요소는 둘러싼 텍스트가 짧은 인라인 인용문이라는것을 나타냅니다.

mark tag

mark tag 란?
HTML <mark> 요소는 현재 맥락에 관련이 깊거나 중요해 표시 또는 하이라이트한 부분을 나타냅니다.
